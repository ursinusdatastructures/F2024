Software Environment

NOTE: If you have previously installed anaconda python 3.9, 3.10, or 3.11 for another course like CS 372, you will be fine to use that and do not need to install anything new.

We will be using a distribution of python known as Anaconda Python, which comes shipped with the majority of the tools that we need for the course. Click here to download Anaconda. Choose the 64-bit version of Python 3.12 for your operating system (Windows/Mac/Linux), download it, and follow the default options during installation, including registering this as the default Python for your system.

Once you install Anaconda, you should be able to launch Spyder and Jupyter, which are the two environments we will be using to write and run code. Under Windows, you can launch these programs by searching for them in the search bar. Under Mac/Linux, you can run them by typing spyder and jupyter, respectively, in the command prompt.
