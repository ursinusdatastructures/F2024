Week 5: Longest Common Subsequence

Chris Tralie

The longest common subsequence (LCS) problem seeks to find the string with the most characters that's in common between two other strings, possibly skipping characters in the sequence. For example, "rinsed a grease" is the LCS between

We rinsed all grease off

000111111110001111110000

Ursinus college students are great students

0101101000001000000100001100111110010001000

We know there are 2N unique binary strings of length N, so a brute force algorithm would be exponential at best. Instead, we're going to make a recursive operation to work backwards.
